# UIImageViewExtension

This simple extension adds loadFromUrl() method to UIImageView class

## Usage:
```Swift
exampleImage.loadFromUrl(NSURL(string: "https://developer.apple.com/assets/elements/icons/128x128/swift.png")!)
```
